# EffectiveMobileCPPTestTask
Тестовое задание на позицию Junior C++ Developer

После клонирования репозитория нужно произвести сборку программы командой **make**.
Чтобы запустить программу после сборки, нужно ввести команду **make run**.
Программу можно запустить вручную, используя команду (если вы находитесь в корне проекта) **./build/app <path_to_.in_file> <path_to_.out_file>**
Входной файл **inlet.in** находится в папке **input**. Вы можете использовать свой файл, однако, если название отличается от **inlet.in**, программа, запущенная с помощью **make run** выдаст ошибку. В таком случае надо подправить таргет **run** в **Makefile** указав правильное название файла или запускать программу вручную.
